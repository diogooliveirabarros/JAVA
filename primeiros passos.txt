3-Verificar pelo cmd se localiza o java e o javac
Se não localizar precisa incluir nas variáveis do windows, em Path e depois adicionar onde se encontra o JDK -> BIN

4-Instalar extensão no VS CODE do JAVA que é da Microsoft, depois disso aperta Ctrl+Shift+P e digita create project java,
seleciona a pasta e dá um nome ao projeto.

5-Para melhor visualização de código instalar o DRACULA ou OMNI, instalei o OMNI.

6-Instalar plugin do JAVA IDE para facilitar a criação de código.

7-Ir até extensões em Debugger for Java e na engranagem 'Configurações' e 'Configurações de Extensão', e mudar para
'Internat Console'

8-Todo código executado aparecerá no 'Console de Depuração' de uma forma mais simples

9-Extensão 'Material Icon Theme' deixa o desenho das pastas bonitinho no VS CODE

10-Extensão 'Extension Pack for Java' cria facilmente os métodos GET e SET
